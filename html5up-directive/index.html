Customer Churn Classification - Machine Learning

In this project, I built and evaluated machine learning models to predict customer Churn. I built logistic regression, XGBoost, and random forest models and performance metrics such as precision, recall, and AUC to evaluate the effectiveness of the models. I selected the model with the best performance on the evaluation metrics and also analyzed the feature importances to understand which factors were most predictive of churn.

Telecom Company Performance Analysis- Data Visualization

I created a visualization using Tableau for a Telecom company to analyze the company's performance with the rollout of 5G. The visualization compares key metrics such as revenue, active users, and average revenue per user before and after the implementation of 5G, as well as the company's market share in relation to its competitors. This visualization provided valuable insights for the company, enabling them to understand the impact of 5G on their business and make informed decisions about their future strategy. It was a challenging but rewarding project, and I enjoyed using my data visualization skills to help the company succeed.

Scraping the real estate Data - Web Scraping

The purpose of this project is to gather data on real estate properties listed on the MagicBricks website for the purpose of analysis. The script uses web scraping techniques to extract the data from the website and save it in a CSV file. The extracted dataset includes the following information for each property: Property_details, Area,Price, Area Type, Number of Square feet are extracted from the website.The script uses Beautiful Soup and Python to scrape the website and extract the data.
